# Enumeration-Algorithm-for-Bayesian-Network
Can handle queries of type : 
a) P(A,B)
b)P(A|B)
c)P(A,B|C)
d)P(A,B|C,D)
e)EU(A,B)
f)EU(A|B)
g)EU(A,B|C)
h)EU(A,B|C,D)
a)MEU(A,B)
b)MEU(A|B)
c)MEU(A,B|C)
d)MEU(A,B|C,D)
If decision nodes exist, they should be mentioned either in queries or evidence. 
Referred Enumeration Algorithm from AIMA.
